## Spacing Scale

Declared values (all multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, badge padding, inline micro-spacing |
| sm | 8px | Button inner padding (vertical), tight element spacing |
| md | 16px | Default element spacing, card internal padding |
| lg | 24px | Section horizontal padding (mobile), card gap |
| xl | 32px | Section horizontal padding (desktop), content block gap |
| 2xl | 48px | Section vertical padding (mobile) |
| 3xl | 64px | Section vertical padding (desktop), major section breaks |
| 4xl | 96px | Hero top/bottom padding |

Exceptions:
- Touch targets (CTA buttons, mobile tap areas): minimum 44px height (accessibility requirement, WCAG 2.5.5)
- Glassmorphism card blur backdrop: `backdrop-blur-md` (12px) — not a spacing value, visual only

## Typography

| Role | Size | Weight | Line Height | Usage |
|------|------|--------|-------------|-------|
| Body | 16px | 400 (Regular) | 1.6 | Paragraph copy, FAQ answers, feature descriptions |
| Label | 14px | 500 (Medium) | 1.4 | Badge text, caption, metadata, price secondary text |
| Heading | 24px | 700 (Bold) | 1.3 | Section headers ("What's Inside", "Who This Is For") |
| Display | 48px / 64px desktop | 800 (ExtraBold) | 1.1 | Hero headline only |

**Font family:** `Inter, -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif`

**Notes:**
- Body: 1.6 line-height (slightly over standard 1.5) — sales copy benefits from generous leading
- Display: 48px on mobile (375px), 64px on desktop (1440px) — responsive via Tailwind `text-5xl md:text-7xl`
- Max 2 weights in non-display use: 400 for body, 700 for headings. Display uses 800 only for the hero H1.
- Price (`$2.99`) uses Display size at weight 800, cyan color — standalone visual prominence

|------|-------|-------|
| Dominant (60%) | `#0A0E1A` dark navy | Page background, hero background, base surface |

**Accent reserved for (explicit list):**
- Primary CTA button background (`Buy Now for $2.99`, `Get Instant Access`)
- Price display text (`$2.99`)
- Active link/focus ring outlines
- Hero headline gradient (cyan → teal direction, text-gradient only on H1)
- Section header accent underline / divider stroke

**Teal reserved for (explicit list — decorative only, not UI affordances):**
- Animated hero background particle/gradient blobs
- Logo color (as-is from asset)
- Glow ring animation around logo

**Lavender reserved for:**
- "What's Inside" chapter icon color
- Star icons in social proof
- Badge background ("Crypto Beginner Guide", "Instant PDF Download")

**Gold: PROHIBITED.** Zero usage anywhere. Source: REQUIREMENTS.md Out of Scope + PROJECT.md constraint.

## Animation Contract

All animations serve conversion — they direct attention, create premium feel, do not distract.

| Element | Animation | Duration | Easing |
|---------|-----------|----------|--------|
| Hero background blobs | CSS keyframes, slow morph + drift | 8s loop | ease-in-out |
| Hero logo | Fade-in + scale from 0.9 | 600ms | ease-out |
| Hero headline | Fade-in + slide-up 16px | 700ms, 100ms delay | ease-out |
| Hero CTA button | Fade-in | 800ms, 200ms delay | ease-out |
| CTA button hover | Glow intensity increase + scale 1.02 | 150ms | ease-out |
| Section entry | Fade-in + slide-up 24px on scroll (IntersectionObserver) | 500ms | ease-out |
| Card hover | Border opacity lift + subtle glow | 150ms | ease-out |
| FAQ accordion | shadcn default expand/collapse | 200ms | ease-out |

**No parallax. No horizontal scrolling. No autoplay video.** Impulse buy = fast load, minimal layout shift.
